If you want to make your websites stop looking like they came from the 1990's, you'll need to gain an understanding for at least the best practices of design and User Experience (UX). This project, breaks apart the website for a popular design magazine <a href="https://smashingmagazine.com" target="_blank"> Smashing Magazine </a>. The goal is to create a grayscale heat map of the website that indicates which elements have the greatest weight in terms of visual hierarchy, typography and design principles.
This project was completed for demonstration purposes in line with standards provided by <a href="https://microverse.org/" target="_blank">Microverse</a>.

## Installation:

> Clone the repository to your local machine

```sh
$ git clone https://github.com/johnsonsirv/smashing-magazine-design-teardown.git
```
### Usage:

> navigate ```cd``` into the cloned directory

```sh
$ cd smashing-magazine-design-teardown
```

> open index.html with your web browser.
